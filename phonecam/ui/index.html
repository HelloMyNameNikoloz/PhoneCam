First run setup

  1. Install or repair the PhoneCam camera device.
  2. Enable Developer Options and USB Debugging on Android.
  3. Connect the phone with USB and accept the debugging prompt.
  4. Allow camera permission in the PhoneCam Android companion.
  5. Confirm the preview is live, then select PhoneCam in OBS or your video app.
PhoneCam is local USB only by default. It does not use cloud services, save recordings, or collect telemetry.

PhoneCam

PhoneCam bridges Android camera frames into a registered Windows camera device.

Close the PhoneCam control window to keep it running in the system tray. Quit from the tray menu when finished.

Open source under GPLv3. No watermark, no account, no subscription.
